== Description == 

WL Tweet List Widget provides easy and quick use in your blog. You can customize the Twitter tweet list in an easy way.

Tweet List Widget. Tweet List Widget is a social plugin that enables Twitter users to attract and gain Retweets & Quotes from their own website.

<strong>Features</strong> 


* Very Easy to install
* Eazy customized layout

== Installation ==

This section describes how to install the plugin and get it working.

1. Download [WL Tweet List Widget](https://wordpress.org/plugins/wl-tweet-list-widget/) plugin.   
2. Upload the WL Tweet List Widget plugin from your admin panel.    
3. Activate the WL Tweet List Widget plugin. 
4. Add WL Tweet List widget to your sidebar.
